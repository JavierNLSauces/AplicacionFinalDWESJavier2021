Metodo validarUsuario()

Metodo que valida si existe un determinado usuario y password en la base de datos. Si existe el usuario actualiza la ultima conexion y el numero de conexiones de ese usuario y lo devuelve. Si no existe el usuario devuelve null.

Parámetros
string$codUsuariocodigo del usuario
string$passwordpassword del usuario
Devuelve
mixed[] Si existe, un array con un objeto de tipo Usuario con los datos de la base de datos y la fechaHoraUltimaConexionAnterior. Si no existe null.
